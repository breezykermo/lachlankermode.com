Reverb

Reverb is a simple way to generate a website from a Google Drive folder. After you connect, you can select a folder and use the documents in it as content for a site.

Select the folder in your Drive to publish (search by name).
You can create relative links between pages on the site by adding a link in a Google Doc to
##{DOC_TITLE}, where DOC_TITLE is the exact name of another document in the same folder.
